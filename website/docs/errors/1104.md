# 1104

## Error message: Missing name for starter page

## How you got here

A starter configuraion entry is missing a filename.
The file might look something like this:

##### file: starter.yaml
```
:
  Type: page
  Folder: /
```

## What to do about it

Give it a name, like this:
```
About:
  Type: page
  Folder: /
```
